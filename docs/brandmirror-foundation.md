# BrandMirror Foundation

## Product Thesis

BrandMirror is a standalone product by SAHAR Studio.

It should feel like a productized brand intelligence tool:

- sharper than a checklist
- more editorial than a dashboard
- more commercially useful than a mood-led audit PDF

The emotional target is premium, calm, intelligent, cinematic, and slightly
severe.

## Current Product Model

BrandMirror should launch as:

- a free first-read product experience
- a paid full report priced at $69
- an optional post-report path into deeper SAHAR work

This keeps the product easy to try while still preserving a clear standalone
commercial offer.

## Free First Read

The free layer should prove that BrandMirror sees something real.

Included in free:

- brand snapshot
- 3 quick scores:
  clarity, premium perception, cohesion
- strongest signal
- main friction
- one next move

Not included in free:

- full positioning analysis
- full tone diagnosis
- full visual identity audit
- friction map
- priority matrix
- rewrite suggestions
- PDF export

## Full Report: $69

The paid layer should feel like a real mini-product, not just a longer version
of the free teaser.

Included in paid:

- brand snapshot
- diagnostic scorecard
- positioning read
- tone check
- visual identity read
- above-the-fold audit
- mixed signals
- friction map
- priority fixes
- 3 rewrite suggestions
- PDF export

Recommended scope:

- one website
- homepage-first analysis
- one primary brand surface only

## Landing Page Structure

1. Hero
   Lead with the product promise and the free try layer. Primary CTA should
   open the first read flow.
2. What BrandMirror reads
   Positioning, tone, and visual identity explained as one system.
3. How it works
   Free first read, then paid unlock for $69.
4. Full report value
   Show what becomes available after purchase.
5. Offer structure
   Free layer, paid report, optional deeper SAHAR follow-through.
6. Final CTA
   Reinforce that the first read is free and the paid report is the true
   product.

## Product Flow

1. User lands on homepage.
2. User clicks `Read my brand`.
3. User enters website and receives free first read.
4. User sees the paywall for the full report.
5. User pays $69.
6. User completes a lightweight intake.
7. User receives the full BrandMirror report.
8. Optional upsell into SAHAR work appears only after the report is delivered.

## Product UX Direction

The result experience should feel like a strategist's read formatted through a
product lens.

Desired output characteristics:

- structured sections rather than long undifferentiated prose
- clear scores or signal markers used sparingly
- explicit friction map
- precise recommendations tied to commercial impact
- typography and layout that support trust and calm authority

Avoid:

- generic AI summary language
- cluttered charts
- playful SaaS widgets
- overusing cards for every idea

## Brand Rules

- SAHAR Studio should remain secondary
- no purple
- use warm ivory, charcoal, stone, muted gold, and muted moss
- strong serif-led editorial hierarchy with restrained sans support
- minimal chrome and deliberate whitespace
