# BrandMirror Report Spec

## Free First Read

Purpose:
Give the user a sharp first impression so they trust the diagnosis enough to
pay for the full report.

Sections:

1. Brand snapshot
2. Quick scores:
   clarity, premium perception, cohesion
3. Strongest signal
4. Main friction
5. One next move

## Full BrandMirror Report ($69)

Purpose:
Turn the first impression into a decision tool the buyer can act on
immediately.

Sections:

1. Brand snapshot
2. Diagnostic scorecard
3. What your brand is signaling
4. Positioning read
5. Tone check
6. Visual identity read
7. Above-the-fold audit
8. Mixed signals
9. Friction map
10. Priority fixes
11. Rewrite suggestions:
    hero line, subheadline, CTA / offer line
12. Strategic next move

## Pricing Logic

Why $69 works:

- low enough to be an impulse diagnostic for founders
- high enough to position the report as a product, not a freebie
- justified by concrete recommendations, not only commentary

## Product Ladder

- Free first read
- $69 full BrandMirror report
- optional SAHAR follow-through after delivery
